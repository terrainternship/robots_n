1. Картинка - представление о проекте
2. Файл buildozer.spec - спецификация для сборки согласно картинки
3. Файлы setup.py или pyproject.toml:
один из них - некие инструкции для сборки, то что они нужны узнал на последней стадии сборки.

Примечание:
Обязательно указывать версию сборки и, чтобы buildozer увидел файл main.py,
в файле buildozer.spec необходимо выбрать (обязательно!):

# (str) Application versioning (method 2)
version.regex = __version__ = ['"](.*)['"]
version.filename = %(source.dir)s/main.py

а в файле main.py добавить сверху переменную типа (это версия):
__version__ = "0.16.5"
